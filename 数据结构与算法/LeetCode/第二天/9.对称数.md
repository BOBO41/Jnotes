## 题目描述

**题目：9.对称数**

![1549290995107](/home/hdr/.config/Typora/typora-user-images/1549290995107.png)

给你一个整数，判断它是否是一个对称数。

## 解题思路

其实就是上一道题目，整数反转的简化版本（因为不用考虑溢出问题）。


## 代码实现

```
public boolean isPalindrome(int x) {
   if (x < 0) return false;
   int result = 0;
   int orig = x;
   while (x != 0) {
      result = result * 10 + x % 10;
      x = x / 10;
   }
   return result == orig;
}
```

# 基本的CRUD

## SELECT

SELECT的基本语法

```sql
SELECT select_list
FROM table_list
WHERE row_constraint
GROUP BY grouping_columns # GROUP BY sex;
ORDER BY sorting_columns # ORDER BY last_name DESC, first_name ASC;
HAVING group_constraint
LIMIT count;
```

## Redis主从复制

主库配置文件

```
# redis-master-6379.conf
port 6379
daemonize yes
logfile "6379.log"
dbfilename "dump-6379.rdb"
dir "/data"
```

从库配置文件

```
# redi-salver-6380.conf
port 6380
daemonize yes
logfile "6380.log"
dbfilename "dump-6380.rdb"
dir "/data"
slaveof redis-master 6379
```

配置网络

```
docker network create -d bridge redis-network
```

创建主库容器

```
docker run -dit --name redis-master --mount type=bind,source="$(pwd)"/redis-master-6379.conf,target=/redis/redis-master-6379.conf --network redis-network redis
```

创建从库容器

```
docker run -dit --name redis-salver1 --mount type=bind,source="$(pwd)"/redis-salver-6380.conf,target=/redis/redis-salver-6380.conf --network redis-network redis
```

# 获取数据库元数据

## SHOW语句

```sql
SHOW DATABASES;
SHOW CREATE DATABASE db_name;

SHOW TABLES;
SHOW TABLE STATUS;
SHOW CREATE TABLE table_name;

SHOW COLUMNS FROM table_name;
SHOW INDEX FROM table_name;
```

## INFORMATION_SCHEMA

`INFORMATION_SCHEMA`是一个数据库。

查看该数据库具有哪些表

```sql
SHOW TABLES IN INFORMATION_SCHEMA;
```
